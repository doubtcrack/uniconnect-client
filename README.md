# UniConnect - Bridging the Gap in Campus Trading

UniConnect is a web application platform designed to provide a convenient way for university students to buy, sell, and exchange their used textbooks, stationeries, and other related materials. The platform aims to bridge the gap in campus trading by enabling students to connect and interact with each other based on their needs and preferences. UniConnect not only facilitates affordable options for students but also promotes sustainability and reduces waste on university campuses.

## Features

- Buy, sell, and exchange textbooks, stationery, and related materials
- Connect with other students based on needs and preferences
- Seamless user authentication and registration using Appwrite
- Real-time chat functionality with Socket.io integration
- Stunning UI designs using Tailwind CSS and Material UI
- Efficient state management with Redux
- Securely store and retrieve data using MongoDB with Node.js
- Host product images and files with Cloudinary integration

## Tech Stack

- Frontend: React.js, Tailwind CSS, Material UI
- Backend: Node.js, Express.js
- Database: MongoDB
- Authentication: NodeJS
- Blogs Manage: AppWrite
- Real-time Communication: Socket.io
- File Hosting: Cloudinary

## Getting Started

1. Clone the repository: git clone https://github.com/doubtcrack/uniconnect-client.git
2.  Install dependencies:
    cd uniconnect-client
    npm install
3. Start the development server: npm start
